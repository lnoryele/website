## Deployment checklist

1. Upload the contents of this folder to the Cloudflare deployment root.
2. Confirm the homepage, robots.txt, sitemap.xml, llms.txt, and 404 behavior live.
3. Add the sitemap in Google Search Console and Bing Webmaster Tools.
4. Submit the included IndexNow key and URL list to Bing IndexNow.
5. Run live Lighthouse/PageSpeed tests after Cloudflare finishes deployment and caching.
6. Monitor Search Console indexing and Core Web Vitals.
