Contact form:
The Request a Briefing form submits through FormSubmit to info@oryele.com. On the first live submission, FormSubmit will send a one-time activation email to info@oryele.com. Confirm that email once; subsequent submissions will be delivered automatically.


CONTACT FORM
The live contact form submits to info@oryele.com through FormSubmit when the website is hosted over HTTP/HTTPS. If contact.html is opened directly from Finder as a local file, the form opens the visitor's email application with a pre-filled message instead. For full live-form testing, run a local web server or deploy the site.
